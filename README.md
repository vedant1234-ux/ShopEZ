## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/shop-ez.git
   cd shop-ez
   ```

2. **Install dependencies**
   ```bash

   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
   Open [http://localhost:5173](http://localhost:5173) in your browser.
